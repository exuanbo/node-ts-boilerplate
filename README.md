# node-ts-boilerplate

[![Renovate enabled](https://img.shields.io/badge/renovate-enabled-brightgreen.svg)](https://renovatebot.com/)

## What's included?

- TypeScript v4
- ESLint with Prettier and Standard style
- Rollup with TypeScript plugin
- All of their configurations
