# @exuanbo/ts-starter

## Usage

- Simply click `Use this template`
- Or write a shell function as example below

```bash
function ts_starter() {
    git clone https://github.com/exuanbo/ts-starter.git $1
    cd $1
    rm -rf .git
    git init
}
```
